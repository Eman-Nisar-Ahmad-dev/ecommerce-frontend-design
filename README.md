# 🛍️ Ecommerce Frontend Design

A responsive multi-page Ecommerce website built using **HTML5**, **CSS3**, and **Vanilla JavaScript**. This project recreates a modern ecommerce UI from a Figma design with clean layouts, responsive design, interactive components, and smooth user experience.

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/Eman-Nisar-Ahmad-dev/ecommerce-frontend-design.git
```

2. Open the project folder.

3. Open **index.html** in your browser.

No installation or build tools are required.
